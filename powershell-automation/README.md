# ⚡ PowerShell Automation Lab

## 📌 Objective

The purpose of this lab was to build and test PowerShell scripts that automate common IT administration tasks.

This project demonstrates how PowerShell can be used to reduce manual work, improve consistency, and support day-to-day IT operations in Microsoft and Windows environments.

---

## 🧩 Scenario

In many IT environments, repetitive administrative tasks can take up valuable time and increase the risk of human error.

As an IT Administrator, I used PowerShell to automate routine support and reporting tasks such as user management, system checks, and administrative reporting.

---

## 🛠️ Technologies Used

- PowerShell
- Windows 10/11
- Windows Server
- Microsoft 365 Lab Environment
- Hyper-V

---

## ⚙️ What I Built

### 1. User Administration Scripts
Created PowerShell scripts to support common account management tasks such as:

- Creating user accounts
- Updating user details
- Managing account-related admin tasks

---

### 2. Reporting Scripts
Built scripts to gather and output useful administrative information, such as:

- User account details
- Device or system information
- Basic administrative reports

---

### 3. Troubleshooting & Support Tasks
Used PowerShell to assist with common support activities, such as:

- Checking system configuration
- Running administrative commands
- Speeding up repetitive support processes

---

### 4. Automation Practice
Developed scripts with a focus on:

- Reducing repetitive manual steps
- Improving consistency
- Making admin tasks more efficient
