# 📱 Intune Endpoint Management Lab

## 📌 Objective

The purpose of this lab was to configure and manage Windows devices using Microsoft Intune, simulating real-world endpoint management in a modern workplace environment.

This includes device enrollment, compliance policies, configuration profiles, and ongoing monitoring of managed endpoints.

---

## 🛠️ Technologies Used

- Microsoft Intune (Endpoint Manager)
- Microsoft 365 Admin Center
- Windows 11 Virtual Machine
- Hyper-V (Lab environment)

---

## ⚙️ What I Configured

### 1. Device Enrollment
- Enrolled Windows 11 device into Microsoft Intune
- Verified successful connection to Azure AD / Microsoft Entra ID
- Confirmed device visibility in Intune portal

---

### 2. Compliance Policies
- Created compliance policies for Windows devices
- Defined security requirements (device health, system settings)
- Configured noncompliance actions

---

### 3. Configuration Profiles
- Deployed configuration profiles to manage:
  - Device settings
  - Security configurations
- Assigned profiles to devices/users

---

### 4. Policy Assignment
- Assigned compliance policies and configuration profiles to:
  - Test devices / users
- Monitored deployment status

---

### 5. Device Monitoring & Validation
- Verified device compliance status
- Checked policy application success
- Reviewed device status in Intune dashboard

---

## ✅ Results

- Device successfully enrolled and managed via Intune  
- Compliance policies applied with correct status (**Compliant**)  
- Configuration profiles deployed without errors  
- Device visible and actively managed in Intune portal
